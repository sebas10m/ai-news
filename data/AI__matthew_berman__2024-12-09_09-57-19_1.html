Setting Up Stable Diffusion 3.5

Stable Diffusion 3.5 is an open source text to image model that can be used to create images like the ones you see here. Today, we will be setting it up using a cloud GPU service called Vulture.

Getting Started with Vulture

To get started, you need to sign up for a Vulture account. It is free to sign up, but you will need to use the code Berman300 to get $300 of credits for free during your first month of use.

Deploying the Cloud GPU

Once you have signed up, go to the left side of the dashboard and click on the Products tab. Then, click on Cloud Compute and select the products tab. Click on deploy, and then select cloud GPU from the options.

Configuring the Cloud GPU

After selecting cloud GPU, you will need to configure the settings. Select the type of GPU you need, and then choose the location that makes sense for you. Select the Ubuntu operating system and the NVIDIA L40S plan. Deselect auto backups and then click deploy.

SSH into the Server

Once the deployment is complete, you will need to SSH into the server. Open up your terminal and enter the command ssh root@ followed by the IP address. Then, enter the password which can be found on the dashboard.

Downloading the Docker Image

Once you are in the server, you will need to download the Docker image. Enter the command docker pull and then download the comfy UI Docker image.

Loading the Docker Image

After downloading the Docker image, you will need to load it. Enter the command docker run and then enter the image name.

Allowing Ports in the Firewall

Next, you will need to allow ports in the firewall. Enter the command ufw allow 8188 and then enter the IP address.

Installing the Hugging Face CLI

Next, you will need to install the Hugging Face CLI. Enter the command python -m pip install huggingface_hub and then enter the command huggingface-cli login.

Loading the Model Files

After installing the Hugging Face CLI, you will need to load the model files. Enter the command huggingface-cli download

Accessing the Comfy UI

Once the model files are loaded, you can access the comfy UI by entering the IP address and the port number in your browser.

That's it. You have now successfully set up Stable Diffusion 3.5 using Vulture. You can now use it to create images like the ones you see here.

Note: All the commands used in this setup process are available in the description of this video. You can use them to set up Stable Diffusion 3.5 on your own Vulture account.
