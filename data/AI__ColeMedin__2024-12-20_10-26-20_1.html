Bolt.DIY Update

Last week, Automator announced its partnership with StackBlitz to make Bolt.DIY the official open source Bolt.New. This significant news has led to an explosion of community involvement, and the project is picking up momentum.

Working on the Roadmap

In addition to the excitement generated by the partnership, a lot of important work has been ongoing to improve the quality of the Bolt.DIY repository.

Upcoming Features

Prominent on the roadmap is the implementation of a prompt library to make it easier to work with different LLMs. Another high priority feature is the addition of a plugin system to make Bolt.DIY super extendable.

Modernizing Bolt.DIY

Other tasks on the roadmap include improving the repository's maturity, creating a more robust and organized codebase, and implementing features like file locking in diffs, a one-click deploy to Netlify, and a Kanban-style board for managing PRs.

Resources for Bolt.DIY

Lots of resources have been created to help users get started with Bolt.DIY, host it themselves, troubleshoot common issues, and more. These resources include guides, videos, and documentation for users to leverage.

New Features in Bolt.DIY

The newest additions to Bolt.DIY are plenty exciting and include features like a new setting modal, experimental system prompts, debug tabs, and much more.

Contributors and the Automator Think Tank

Special recognition goes out to active contributors like Dustin and Anirban, who have been diligently working on several key features.
