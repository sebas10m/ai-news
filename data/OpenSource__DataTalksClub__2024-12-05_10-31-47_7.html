Open Source Spotlight: Kestra

Welcome to Kestra

Kestra is a unified orchestrator that allows you to take loads of different inputs, process them, and get a lot of different outputs. You can do this on a schedule or event-based.

Key Features of Kestra

  • Unified orchestrator
  • Supports YAML and UI configuration
  • Has a VS Code plugin for validation
  • Supports container images
  • Has a key-value store for statefulness
  • Supports AWS integrations
  • Has a simple and intuitive UI

How Kestra Works

Kestra works by allowing you to create workflows that can be triggered by events or schedules. These workflows can be composed of different tasks, which can be executed in parallel or sequentially.

Contributing to Kestra

Kestra is an open-source project and welcomes contributions from the community. We have a GitHub repository where you can find issues to contribute to, and a Slack channel where you can ask questions and get help.

So, if you're interested in contributing to Kestra, don't be afraid to reach out and get started!
