Advanced Prompt Engineers for Large Language Models

Getting the prompt right is hard, especially if you want to deliver consistent results at scale. But what if you can control the prompt to get specific output structure and general image and charts with just one single prompt?

Guidance Framework

The first framework I want to talk about is called Guidance. It is an open-source framework initially introduced by Microsoft and has more than 12,000 stars on GitHub. Fundamentally, it is a framework that allows you to program the prompt to get specific output.

  • Define a specific output structure you want.
  • Restrict the actual output that Large Language Model generates.
  • Set up advanced logic like if conditions.

Using Guidance for Advanced Prompt Engineering

With the Guidance framework, you can achieve complex tasks like:

  • Restricting the output of Large Language Model to predefined answers.
  • Setting up advanced logic like if conditions.
  • Generating real-time charts and images.

Other Communities and Open Source Projects

While the Guidance framework is powerful, it still has limitations. To speed up the process, I recommend exploring other communities and open-source projects, such as:

  • FlowGPT: a prompt library and community.
  • Prompts Royal: a front-end based on GPT prompt engineer.

These resources can help you discover better starting points for your prompts and evaluate different variants of prompts to get the best results.
