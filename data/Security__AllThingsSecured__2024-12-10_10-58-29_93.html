Setting Up Two-Factor Authentication on Twitter

Twitter has removed SMS two-factor authentication as an option for nonTwitter Blue subscribers, but this is an opportunity to increase your security by learning how to use an authenticator app or security keys.

Using an Authenticator App

You can use an authenticator app, such as Google Authenticator, to generate two-factor authentication codes. To set up an authenticator app, follow these steps:

  • Download and install an authenticator app on your phone.
  • Open the app and click on "Get started."
  • Scan the QR code provided by Twitter using the app.
  • Enter the 6-digit code generated by the app.
  • Save the backup code provided by Twitter.

Using a Security Key

You can use a security key, such as a YubiKey, to add an extra layer of security to your two-factor authentication. To set up a security key, follow these steps:

  • Plug in the security key and tap it to initialize it.
  • Follow the prompts to add the security key to your Twitter account.
  • Save the backup code provided by Twitter.

It's recommended to set up multiple methods of authentication, such as an authenticator app and a security key, to add an extra layer of security to your Twitter account.

Conclusion

Setting up two-factor authentication on Twitter is an important step in securing your account. You can use an authenticator app or a security key to generate two-factor authentication codes. Remember to save the backup code provided by Twitter and consider setting up multiple methods of authentication for added security.
