Setting Up Your Ethical Hacking VM

In this video, we will be setting up your ethical hacking VM and configuring it to make your workflow as efficient as possible.

Step 1: Installing OH MY ZSH

OH MY ZSH is a community-driven framework for managing zsh configurations. It adds a lot of features and functionality to your terminal, including theming, plugins, and syntax highlighting.

We will be installing OH MY ZSH and setting it up to work with our Kali Linux VM.

Step 2: Installing Plugins

We will be installing several plugins to enhance our terminal experience.

  • Easy A: a modern plugin for LS that colorizes and makes it easier to read
  • Auto Suggestions: a plugin that shows you previously used commands and their flags
  • GRC: a plugin that adds color to your end-map and ping commands
  • Tmux: a plugin that allows you to customize your tmux configuration
  • ZSH Syntax Highlighting: a plugin that adds color to your zsh syntax
  • LS Colorize: a plugin that adds color to your ls output

Step 3: Customizing Tmux

We will be customizing our tmux configuration to make it more efficient and user-friendly.

  • Binding control A as our prefix key instead of control B
  • Unbinding the default control B command
  • Setting the initial window to 1 instead of 0
  • Using pipe and minus to split panes
  • Adding a line to allow renaming of windows
  • Adding a line to allow use of alt keys

Step 4: Installing Batcat

Batcat is a plugin that allows us to view our files with syntax highlighting.

We will be installing Batcat and setting it up to work with our Kali Linux VM.

Step 5: Adding Aliases

We will be adding several aliases to our.zshrc file to make our commands more efficient and user-friendly.

  • Aliasing cat to back cap
  • Aliasing nmap to grc nmap
  • Aliasing ls to ls colorize

Step 6: Installing Nerd Fonts

Nerd Fonts is a font that includes more icons and better styling for our terminal.

We will be installing Nerd Fonts and setting it up to work with our Kali Linux VM.

Conclusion

That's it for this video! Our Kali Linux VM is now set up with OH MY ZSH, plugins, and aliases to make our workflow more efficient and user-friendly.
