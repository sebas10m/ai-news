Selling 3D Models on Colts.com

In today's online tutorial, I'm going to show you how to make money selling simple 3D models on Colts.com. We will cover how to generate 3D models using AI tools, upload them to Colts, and set up a passive income stream.

Generating 3D Models using AI Tools

We can use AI tools like Tripod 3D AI to generate 3D models instantly. The website provides free credits, and we can export the models in STL, OBJ, or other formats.

Uploading 3D Models to Colts

Once we have our 3D model, we can upload it to Colts.com. We will need to provide a description, add images, and set a price for the model.

Pricing and Revenue Sharing

Colts.com takes a 15% commission, and the bank takes 5% of the net selling price. We will receive 80% of the net selling price.

Let's follow along with the tutorial and see how to set up a passive income stream on Colts.com.

Step-by-Step Guide

  1. Generate a 3D model using AI tools like Tripod 3D AI.
  2. Export the model in STL or OBJ format.
  3. Upload the model to Colts.com.
  4. Provide a description and add images.
  5. Set a price for the model.
  6. Choose a revenue sharing model and confirm the listing.

By following these steps, we can set up a passive income stream on Colts.com and start earning money selling 3D models.
