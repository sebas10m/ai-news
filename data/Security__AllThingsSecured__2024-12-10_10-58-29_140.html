Setting Up a VPN on Chromebook

There are four different ways to set up a VPN for Chromebook, and here's a step-by-step guide on how to do it using my Chromebook as an example.

Method 1: Android App

If you have a newer Chromebook, the easiest way to set up a VPN is to use an Android app. This will encrypt your entire connection, not just what's on your browser. To do this, follow these steps:

  • Search for VPN on the Google Play Store.
  • Install a VPN app, such as ExpressVPN.
  • Set up a subscription to the VPN service.
  • Install the app on your Chromebook.
  • Set up a VPN configuration.
  • Press OK to complete the setup.

Method 2: Chrome Browser Extension

If your Chromebook doesn't allow for Android apps, you can use a Chrome browser extension. To do this, follow these steps:

  • Search for NordVPN in the Chrome Web Store.
  • Install the extension.
  • Log in to your account.
  • Click on Quick Connect to establish a VPN connection.

Method 3: Manual Native Setup

If your Chromebook doesn't allow for Android apps and you don't want to use a Chrome extension, you can set up a manual native setup using the OpenVPN L2TP protocol. To do this, follow these steps:

  • Go to the VPN settings on your Chromebook.
  • Click on the plus arrow.
  • Choose L2TP with a pre-shared key.
  • Set up a L2TP connection using your VPN account details.

Method 4: Linux App

If you have set up the Linux app on your Chromebook, you can set up a VPN using a VPN app. To do this, follow these steps:

  • Search for a VPN app in the Linux app store.
  • Install the app.
  • Set up a VPN connection.

Note: This method is beyond the scope of this video, and it's recommended to use one of the other three options.
