Emoji Maker

This is a web application that allows users to generate and display emojis.

  • Frontend built using Next.js and Chassian UI components
  • Backend built using Superbase and Clark authentication
  • Emoji generation using a Replicate model
  • Storage of emojis using Superbase storage

Authentication is done using Clark, which supports various auth methods.

The application also includes features such as:

  • Download image
  • Liking an emoji (user can only like once)
  • Displaying the total count of likes

Deployment is done using Vercel.
