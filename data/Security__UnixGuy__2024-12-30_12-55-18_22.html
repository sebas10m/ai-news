Updated No BS Guide to Becoming a Cloud Security Professional

In this guide, we'll cover the steps to become a cloud security professional, without any prior technical skills or degrees. We'll start with the basics of cloud security and then move on to learn cloud fundamentals and cybersecurity fundamentals.

Step 1: Learn Cloud Security Fundamentals

Start with learning cloud security fundamentals. This will give you a solid understanding of cloud security concepts, including threat detection, incident response, and security monitoring.

Step 2: Learn a Cloud Platform (AWS, Azure, or Google Cloud)

Learn one of the three major cloud platforms: Amazon Web Services (AWS), Microsoft Azure, or Google Cloud. Each platform has its own unique features, so choose one to focus on. I recommend starting with AWS, as it's the most widely used cloud platform.

Step 3: Learn AWS Security

Once you've mastered a cloud platform, learn AWS security in-depth. Start with the AWS Cloud Practitioner exam, which covers cloud fundamentals, security, and compliance. Then, move on to the AWS SysOps Admin exam, which covers deployment, provisioning, and automation, among other topics.

Step 4: Learn Microsoft Azure Security

After mastering AWS security, move on to Microsoft Azure security. Start with the Microsoft Cybersecurity Analyst Certificate, which covers IT, networking, and operating systems. Then, move on to the Microsoft Azure SC 900 exam, which covers security compliance for Microsoft Azure Cloud.

Step 5: Learn Google Cloud Security (Optional)

If you want to learn Google Cloud security, start with the Google Cloud Computing Foundation Certificate, which covers cloud computing fundamentals. Then, move on to the Google Cloud Security Engineer certificate, which covers security engineering for Google Cloud.

Step 6: Get Hired as a Cloud Security Professional

Once you've completed the steps above, it's time to get hired as a cloud security professional. Make sure to understand how cloud security fits within the cybersecurity ecosystem and that job titles don't always match the tasks. Also, be sure to do some practical projects and continue learning as a cybersecurity professional.
