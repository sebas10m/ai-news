SmallAI: A Revolutionary AI Project

Every once in a while, there's a project that made me feel we're just scraping the surface with generative AI. And SmallAI is definitely one of those projects. It got more than 8,000 stars on GitHub in just five days, and it is one of the fastest growing open source AI projects.

What is SmallAI?

SmallAI is an AI that writes codes for you. And it's different from asking ChatGPT to write some simple code snippet. It actually builds out an entire code base for even very complicated apps.

An Example

Here's a quick example. I asked GPT-4 to write a product requirement doc for a classic snake game, and then pass on those requirements to the SmallAI. And asked it to build this app. And boom, I just got this snake game that is fully functional from SmallAI.

How Does SmallAI Work?

When you're looking through their code base, it's actually very straightforward. First, they pass the product requirement into a prompt. They will generate the architecture of the apps, what are the different files we need to create. And then it will pass this list of files. As well as original code.

Tips and Tricks

One tip I want to share is that if you want to use SmallAI to generate code, it's recommended to write a clear and specific product requirement. This will help SmallAI to generate code that meets your needs.

Debugging

SmallAI also provides a way to debug the code. If you add any arrows when running the code, you can create a debug file and run it with the model. This will give you analysis of the potential arrows and how to fix them.

Conclusion

SmallAI is a revolutionary AI project that has the potential to change the way we develop software. While it's still a little bit buggy and not capable to produce very complicated apps out of box yet, I'm excited about its possibilities and look forward to seeing its development in the future.
