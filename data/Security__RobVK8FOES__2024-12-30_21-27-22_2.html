Antenna Rotator Control Tutorial

This tutorial will guide you through controlling an antenna rotator from your computer using a USB IR transceiver.

Step 1: Install WinLIRC and drivers

First, you need to install WinLIRC and the drivers for the USB IR transceiver on your computer.

Once installed, open WinLIRC and configure it to use the IguanaWorks USB IR transceiver.

Step 2: Configure the antenna rotator remote control

Next, you need to configure the remote control for the antenna rotator using the RCA VH2260 configuration file.

You can download the configuration file from the description or pinned comment of this video.

Step 3: Run the Rotator Control application

Once configured, run the Rotator Control application to control the antenna rotator remotely using your computer.

Select the antenna rotator you want to control and choose the bearing you want it to point to.

Step 4: Verify the antenna rotator movement

Verify that the antenna rotator is moving to the chosen bearing using the SDR sharp application or Google Earth.

Step 5: Install and configure LIRC on Linux

For Linux users, install LIRC and the IguanaWorks USB IR transceiver drivers on your system.

Configure LIRC to use the IguanaWorks USB IR transceiver and program the remote control for the antenna rotator.

Step 6: Run the Rotator Control application on Linux

Once configured, run the Rotator Control application using Python and the TK framework to control the antenna rotator remotely.

Enter the arbitrary direction you want the antenna rotator to point to and verify its movement using the SDR sharp application or Google Earth.
