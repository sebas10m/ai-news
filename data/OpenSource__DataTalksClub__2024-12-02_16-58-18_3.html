Open Source Spotlight: Kedro

About Kedro

Kedro is a Python-based data pipelining framework developed by Dipyaman and his team at Voltron Data.

Kedro Features

  • Provides a consistent way of building up initial project structure
  • Less opinionated overall, allowing users to adapt to their own templates and ways of organizing code
  • Has a general structure, but can be configured and customized
  • Supports data catalog concept, abstracting between code logic and data storage
  • Includes a library of data sets and allows users to create their own
  • Easy to unit test and includes automated linting and documentation generation
  • Extensive plugin ecosystem, including support for Kubeflow and Vertex AI

Contributing to Kedro

Kedro has a vibrant user community and welcomes contributions. The best way to get started is to use Kedro and identify areas where you'd like to see improvements or new features.

  • Start by exploring the Kedro documentation and tutorials
  • Join the Kedro community and engage with other users and contributors
  • Fix issues or create new features through pull requests (PRs)
  • Contribute to the Kedro datasets library by implementing new data sets or enhancements
