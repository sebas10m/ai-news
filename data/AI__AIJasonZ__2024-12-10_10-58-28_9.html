Reddit Analytics Platform

Core Functionality

To display a list of available subreddits and add a new subreddit, the following core functionality is required:

  • Display a list of subreddits
  • Add a new subreddit

Data Retrieval

To retrieve data from Reddit, the following data retrieval functionality is required:

  • Fetch Reddit data for a specific subreddit
  • Fetch post data for a specific subreddit

Post Categorization

To categorize posts, the following post categorization functionality is required:

  • Categorize posts based on their content
  • Display categories for each post

Superbase Integration

To integrate with Superbase, the following functionality is required:

  • Initialize Superbase clients
  • Create database tables
  • Save data to Superbase
