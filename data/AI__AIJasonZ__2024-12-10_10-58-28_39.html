Building AI Researcher 3.0: A Step-by-Step Guide

In this article, we will explore how to build an AI researcher 3.0 using a multi-agent system. This system will consist of three different GPT assistants: a director, a research manager, and a researcher. The director will be able to read and update Airtable records and delegate research tasks to the research manager and researcher. The research manager will generate a research plan and review the results from the researcher. The researcher will browse the internet and gather information.

Step 1: Create the Director Agent

  1. Create a new GPT assistant agent with the following special prompt:
  2. You are the director of a research company. You will extract a list of companies to research from Airtable and break them down into individual research tasks.
  3. Delegate each research task to the research manager and researcher.
  4. Update company information individually to Airtable.
  5. Register two functions: get Airtable records and update single Airtable record.

Step 2: Create the Research Manager Agent

  1. Create a new GPT assistant agent with the following special prompt:
  2. You are a research manager. You will generate a research plan and review the results from the researcher.
  3. Propose next research tasks to the researcher.
  4. Register two functions: web scraping and summary.

Step 3: Create the Researcher Agent

  1. Create a new GPT assistant agent with the following special prompt:
  2. You are a world-class researcher. You will browse the internet and gather information on specific research tasks.
  3. Script websites to gather information.
  4. Register three functions: Google search, web scraping, and summary.

Step 4: Connect the Agents Together

  1. Open the Autogen framework and set up a new project.
  2. Create a new file called oai-config-list and add the OpenAI API key and model ID.
  3. Import the necessary libraries and load the environment and config list.
  4. Create a list of functions to use, including Google search, web scraping, and Airtable API end points.

Running the System

  1. Run Python app.py and initiate a chat with the user proxy agent.
  2. Trigger a research task and view the results from the researcher.
  3. Review and critique the results from the researcher.
